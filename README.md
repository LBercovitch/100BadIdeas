# 100 Bad Ideas
A filterable index page of random projects, tools, games, and designs.

## Instalation Instructions

- Clone the repository.
- Navigate to the project's directory: `cd 100BadIdeas/bad-ideas`
- Using npm, install dependancies: `npm i`
- Start the server: `npm run dev`
- In your browser, go to the url shown in the terminal, by default it will be something like this: `http://localhost:5173/`
